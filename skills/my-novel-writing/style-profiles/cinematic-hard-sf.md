# Style Profile: Cinematic Hard SF

## Use When

Use this profile when the user wants broad-scale science-fiction atmosphere with concrete images, technical constraints, and emotional longing.

## Texture

- start close, then widen the frame
- body → object → room/vehicle → vista → wider system → desire
- reveal world through equipment, transit, maintenance, scarcity, surveillance, or cost
- use concrete materials, sound, pressure, light, distance, motion
- keep the world vast but grounded in one POV

## Avoid

- lore dump openings
- abstract grandeur without objects
- generic “advanced civilization” phrases
- technical detail that does not affect action

## Checklist

- [ ] Is there a close physical anchor?
- [ ] Does the wider scale mean something to the POV?
- [ ] Does at least one technology create friction?
- [ ] Does the scene have desire or denial attached to the setting?
