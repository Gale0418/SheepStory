# Style Profile: Quiet Emotional Detail

## Use When

Use this profile for intimate scenes, grief, longing, guilt, hesitation, exhaustion, or fragile trust.

## Texture

- emotion shown through small actions
- object handling
- silence and incomplete speech
- physical habits
- avoidance
- repeated gestures
- environmental details that echo mood

## Avoid

- naming emotions too early
- therapeutic explanations
- characters resolving everything cleanly
- poetic fog that hides the concrete moment

## Checklist

- [ ] What object carries the emotion?
- [ ] What does the character refuse to say?
- [ ] What small action reveals the feeling?
- [ ] What silence should remain unexplained?
