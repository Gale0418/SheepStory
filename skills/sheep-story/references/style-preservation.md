# Style Preservation

## Purpose

Style preservation prevents revision from destroying the author's voice.

Polishing should improve clarity, pressure, rhythm, and specificity without flattening the prose into generic assistant output.

## Protect These

Default to preserving:

- distinctive sentence rhythm
- deliberate repetition
- character-specific word choice
- awkwardness that reveals character
- silence and omission
- ambiguity
- narrator bias
- sensory habits
- recurring motifs
- rough emotional texture
- unusual but intentional phrasing

## Safe to Change

Usually safe:

- typos
- broken punctuation
- accidental repetition
- paragraph breaks that obscure meaning
- unclear pronoun reference
- inconsistent tense when not intentional
- mechanical transition words
- generic summary that can become visible action

## Dangerous to Change

Be careful with:

- dialogue rhythm
- inner monologue
- flirtation
- hostility
- evasive speech
- unreliable narration
- scene-level ambiguity
- intentionally ugly or blunt sentences
- repeated images that may be motif rather than mistake

## Style Snapshot

Before revising a passage, create a short style snapshot:

```markdown

- POV distance:
- Sentence rhythm:
- Dialogue texture:
- Emotional expression style:
- Sensory focus:
- Level of directness:
- Roughness to preserve:
- Things not to smooth away:
```

## Revision Method

1. Identify what the passage is trying to do.
2. Identify which texture carries voice.
3. Fix only what interferes with the scene's effect.
4. Preserve character-bearing imperfections.
5. Run anti-AI-flavour without over-polishing.

## Anti-Sterilization Check

After revision, ask:

- Does this still sound like the same narrator?
- Did the characters become too polite?
- Did ambiguity disappear unnecessarily?
- Did the prose become too evenly paced?
- Did rough emotion turn into elegant explanation?
- Could any assistant have written this version?

If yes, restore texture.


## Principle

Do not turn a living draft into a clean corpse.
