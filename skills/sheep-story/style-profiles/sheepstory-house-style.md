# Style Profile: SheepStory House Style

## Purpose

Apply SheepStory's original house taste only when the user requests it or the project already declares this profile.

## Preferences

- concrete scene texture and strong visual continuity
- hard-SF or military-SF friction
- first-person or close-POV technical reasoning when appropriate
- practical calculations, material limits, equipment constraints, and uncertainty
- dry humor under pressure
- competent but imperfect characters
- worldbuilding revealed through objects, action, cost, scarcity, and longing
- intelligent characters whose reasoning can be reconstructed
- conflict that is not smoothed away too quickly

## Technical Energy

```text
problem → observation → principle → estimate or test → uncertainty → decision → human reaction
```

Use knowledge as a tool under pressure, not as a textbook interruption.

## Scene Movement

When appropriate, move from close physical experience toward wider scale:

```text
body → object → room / vehicle / street → opening / screen / scope → wider world → desire or denial
```

Do not force this sequence onto every opening.

## Avoid

- story-external encyclopedia worldbuilding unless intentionally requested
- decorative science or numbers without consequence
- generic destiny language
- perfect plans, perfect information, or convenient enemy stupidity
- customer-service dialogue
- compulsory conflict where a quiet scene already has dramatic function
- polishing away roughness that carries voice

## Boundary

This profile never overrides supplied prose, current story state, explicit user instructions, or language-specific voice evidence.
