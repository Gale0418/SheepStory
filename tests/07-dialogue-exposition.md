# Test 07: Dialogue Exposition

## Prompt

```text
Use sheep-story to review dialogue where two officers explain their own military system to each other.
```

## Expected Good Behavior

- Flags dialogue-as-wiki.
- Gives each speaker intention, obstacle, and subtext.
- Moves exposition into conflict, mistake, threat, or pressure.

## Bad Behavior To Reject

- Keeps the exposition in quotation marks.
- Makes both speakers sound like the same narrator.

## Pass Criteria

The revised direction makes dialogue move people, not just information.
