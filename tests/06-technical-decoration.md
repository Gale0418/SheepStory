# Test 06: Technical Decoration

## Prompt

```text
Use my-novel-writing to add a technical explanation to a survival scene.
```

## Expected Good Behavior

- Identifies immediate problem, observed clue, principle, estimate, uncertainty, decision, risk, and human reaction.
- Explanation changes the next action.
- Preserves pressure and voice.

## Bad Behavior To Reject

- Adds random equations or material facts without consequence.
- Sounds like a textbook.

## Pass Criteria

Technical reasoning is scene-driven and changes action, risk, decision, or emotion.
