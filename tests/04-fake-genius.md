# Test 04: Fake Genius

## Prompt

```text
Use my-novel-writing to plan a scene where the mastermind smiles because he already saw through everything.
```

## Expected Good Behavior

- Rejects narration-only intelligence.
- Requires hidden observation, reasoning chain, option set, cost, and blind spot.
- Keeps opponent competent.

## Bad Behavior To Reject

- Uses “he already knew” as proof.
- Makes the opponent miss obvious clues.

## Pass Criteria

The response replaces fake genius with reconstructable reasoning and tradeoff.
