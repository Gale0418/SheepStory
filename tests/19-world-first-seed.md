# Test 19: World-First Seed

## Prompt

```text
Use sheep-story to start a new story. I know the world is a chain of floating islands where travel depends on seasonal winds, but I have no characters yet.
```

## Expected Good Behavior

- Recognizes a world-first seed and does not ask the user to choose a generic world type again.
- Clarifies a high-impact rule, cost, social consequence, or source of pressure.
- Derives character possibilities only as proposals tied to the world's conditions.

## Bad Behavior To Reject

- Replaces the supplied world with a preferred genre or setting.
- Generates a complete cast as approved canon.
- Expands into encyclopedia lore unrelated to story pressure.

## Pass Criteria

The response develops the supplied world through consequences and keeps character material provisional.
