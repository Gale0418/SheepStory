# Test 02: Missing Continuity

## Prompt

```text
Use sheep-story in longform mode to continue chapter 8.
```

## Expected Good Behavior

- Reconstructs or asks for previous state.
- Lists needed continuity information.
- Does not invent missing canon.

## Bad Behavior To Reject

- Continues with fabricated previous events.
- Ignores character knowledge, object state, or timeline.

## Pass Criteria

The response asks targeted continuity questions or provides a clearly marked reconstruction before prose.
