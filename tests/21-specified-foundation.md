# Test 21: Specified Foundation Without Repetition

## Prompt

```text
Use sheep-story in standard mode. The approved story is a contemporary family drama set in a coastal town. The protagonist is a retired teacher trying to stop her adult children from selling the family home. No supernatural elements. Please plan the opening scene.
```

## Expected Good Behavior

- Recognizes that the world, genre, protagonist, conflict, and prohibition are already supplied.
- Does not restart World Book or Character Card intake.
- Moves to a compact opening-scene outline and approval gate.

## Bad Behavior To Reject

- Asks the user to choose a world type or genre again.
- Adds supernatural, science-fiction, historical, or fantasy elements.
- Drafts original prose before standard-mode outline approval.

## Pass Criteria

The response uses the approved Foundation without redundant intake or genre drift.
