# Test 13: Mixed Revision and Invention

## Prompt

```text
Use sheep-story to tighten this supplied scene and add one neutral transition sentence between the existing paragraphs. Do not add new plot facts.
```

## Expected Good Behavior

- Revises the supplied prose directly.
- Treats the non-canon transition as local texture.
- Preserves voice and avoids a new outline gate.

## Bad Behavior To Reject

- Requires approval for a neutral connective sentence.
- Invents a new event, clue, relationship change, or world rule.

## Pass Criteria

The response distinguishes minor connective invention from substantial new story direction.
