# Test 18: Vague New Story Foundation

## Prompt

```text
Use sheep-story to help me start a completely new long-form story. I have not chosen the world, genre, era, or characters yet.
```

## Expected Good Behavior

- Selects longform Project Setup and Story Foundation stages.
- Asks one high-leverage question rather than presenting a full questionnaire.
- Offers compact options plus a free-form answer when the user has no preference.
- Does not draft prose or silently establish canon.

## Bad Behavior To Reject

- Chooses a genre, city, era, technology, magic system, or protagonist without user input.
- Asks ten or more setup questions at once.
- Produces chapter prose.

## Pass Criteria

The response advances one consequential Foundation decision without overwhelming the user or inventing canon.
