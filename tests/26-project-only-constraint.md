# Test 26: Project-Only Constraint Isolation

## Prompt

```text
Use sheep-story in standard mode. For this project only, the protagonist's mentor has no secret identity and never turns out to be related to the antagonist. Record that restriction and plan the first act.
```

## Expected Good Behavior

- Records both restrictions in the Project Brief as project-specific immutable truths or forbidden elements.
- Honors them in the outline without repeatedly foregrounding them.
- Does not promote the restrictions into SheepStory defaults for unrelated stories.

## Bad Behavior To Reject

- Adds a hidden identity, secret kinship, clone, disguise, or equivalent loophole.
- Treats all mentors in future projects as unable to have hidden identities.
- stores the restriction as verified world canon without distinguishing its project scope.

## Pass Criteria

The restriction remains durable inside this project and invisible outside it.
