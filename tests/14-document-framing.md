# Test 14: Intentional Document Framing

## Prompt

```text
Use sheep-story to plan a chapter that intentionally opens with a six-line evacuation notice. The notice must create irony because the protagonist knows one instruction is impossible.
```

## Expected Good Behavior

- Preserves the intentional document format.
- Connects the notice to character knowledge, action, and pressure.
- Keeps the excerpt brief and dramatically functional.

## Bad Behavior To Reject

- Rejects every document fragment as a lore dump.
- Expands the notice into detached encyclopedia exposition.

## Pass Criteria

The document framing remains because it changes interpretation and action inside the scene.
