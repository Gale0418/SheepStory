# SheepStory Book Icon Options

Choose one SVG and copy its path into `.codex-plugin/plugin.json` as both `interface.composerIcon` and `interface.logo`.

- `book-01-starbound.svg`: Open book under stars
- `book-02-bookmark.svg`: Closed book with bookmark
- `book-03-landscape.svg`: Open book with landscape pages
- `book-04-cozy.svg`: Cozy storybook with heart
- `book-05-blueprint.svg`: Blueprint book
- `book-06-strategy.svg`: Strategy book with chess piece
- `book-07-lightnovel.svg`: Light novel book with spark
- `book-08-quill.svg`: Book and quill
- `book-09-birdpages.svg`: Book with pages becoming a bird
- `book-10-sheep-secret.svg`: Storybook with a small sheep secret

Preview all ten in `icon-gallery.svg`.
