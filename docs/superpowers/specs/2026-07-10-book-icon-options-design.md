# Book Icon Options Design

Ten standalone SVG options use a 256 by 256 viewBox, flat colors, no external assets, and a book as the primary silhouette. The current plugin icon remains unchanged until the user selects an option.

Each option must parse as XML and remain recognizable at small icon sizes. The concepts cover story, bookmark, horizon, cozy fiction, blueprint, strategy, sparkle, quill, flying pages, and a subtle SheepStory motif.
